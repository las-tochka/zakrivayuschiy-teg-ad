# Закрывающий тег

https://github.com/las-tochka/zakrivayuschiy-teg-ad.git

## Описание проекта
Проект "Закрывающий тег" — это веб-страница-дневник, рассказывающая о процессе обучения на курсе по фронтенд-разработке. Дизайн страницы выполнен в стилистике компьютерных интерфейсов с возможностью переключения тем, анимациями и интерактивными элементами.

## Особенности
### Темы
Темная тема — для ночного использования
Светлая тема — для дневного использования
Автоматическая тема — следует системным настройкам
Переключение через меню в шапке сайта

### Интерактивные элементы
Кнопки лайков с анимированными SVG-иконками
Диалоговое окно "Сохранить на память" с дискетой
Адаптивный дизайн для различных устройств
